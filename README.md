# D2R Holidays 🌴✈️🏖️

**D2R Holidays** - Your premier travel agency website offering customized holiday packages, tour planning, and instant booking services for leisure, adventure, and cultural trips across domestic and international destinations.

## 🎯 Project Overview
This repository contains the complete source code for the live website: **[https://d2rholidays.com](https://d2rholidays.com)**

**What we offer:**
- Personalized holiday packages (beach, hills, wildlife, culture)
- Complete travel solutions (flights + hotels + tours)
- Domestic & International destinations
- Budget to luxury packages for every traveler

## ✨ Key Features
- 🏝️ **Curated Holiday Packages** - Beach escapes, hill stations, adventure tours
- ✈️ **Online Booking System** - Flights, hotels, transfers, guided tours  
- 🧳 **Custom Trip Planning** - Dedicated travel experts
- 📱 **Fully Responsive** - Mobile, tablet, desktop optimized
- ⭐ **Customer Testimonials** - Real traveler stories
- 💬 **Contact Forms** - Instant inquiry responses
- 🔒 **Secure & Fast** - SSL protected, quick loading

## 🚀 Live Demo
**Live Website:** [https://d2rholidays.com](https://d2rholidays.com)

## 🛠️ Technology Stack

Frontend: HTML5, CSS3, JavaScript, Bootstrap 5
Backend: PHP / WordPress (booking system)
Database: MySQL / MariaDB
Styling: Bootstrap + Custom CSS
JavaScript: jQuery + Custom Scripts
Hosting: Custom domain with SSL certificate

## 📋 Quick Start (Local Development)
```bash
# 1. Clone repository
git clone https://github.com/YOURUSERNAME/d2r-holidays.git
cd d2r-holidays

# 2. For Static HTML (no server needed)
# Simply double-click index.html or drag to browser!

# 3. For PHP/WordPress setup
# Install XAMPP/WAMP/LAMP
# Copy files to htdocs/www folder
# Start Apache + MySQL
# Visit: http://localhost/d2r-holidays

# 4. NPM setup (if using modern frontend)
npm install
npm run dev

Local URLs:

    Static: file:///path/to/index.html

    PHP: http://localhost/d2r-holidays

    Dev server: http://localhost:3000
